# Track Location and Provider Using Phone Number

This project is a simple Python application that tracks the geographical location and telecom service provider of a phone number using the `phonenumbers` library.

## Features

- Detects the country or region of a phone number
- Identifies the telecom carrier or service provider
- Uses the `phonenumbers` library for accurate parsing
- Simple and beginner friendly Python project

## Technologies Used

- Python
- phonenumbers library
